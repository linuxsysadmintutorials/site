disqus_title: Install puppet syntax highling for vim


Install the vim-puppet package.

```
$ sudo apt-get install vim-puppet
```

Enable the vim-puppet module globally.

```
$ sudo vim-addons -w install puppet
```

Or alternatively you can enable it for your user account.

```
$ vim-addons install puppet
```
